What does it do
===============
This script will directly compare two mbn's together

Usage
=====
For basic usage:
python mcfg_mbn_comparison.py --left tmo1.mbn --right tmo2.mbn

For more help with options + features:
python mcfg_mbn_comparison.py --help


Installation
============
1) Place \\fosters\GobiFW\Binmo\Tools\mcfg_mbn_comparison folder
in modem_proc\mcfg\mcfg_tools

2) Place \\fosters\GobiFW\Binmo\Tools\mcfg_translate
in modem_proc\mcfg\mcfg_test
(if this folder is already in CRM, replace it for
parsing HW mbn support)
